# 项目简介

## 核心功能
- 前后端代码
- SQL文件
- 万字文档
- 参考导入调试视频
- 讲解视频
- 开发工具安装包

## 适用场景
- 管理员登录查看主页、飞行器、维护人员、个人简历、职位类型、安全等级评定、应聘信息、面试信息、推送消息、公告信息、论坛分类、举报记录等信息

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
